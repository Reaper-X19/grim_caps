# Grim Caps Website

## Vision

A jaw-dropping, minimal, and elegant 3D immersive website that showcases the infinite power of customization for mechanical keyboard keycaps. The site should feel premium, sophisticated, and highlight the artistry of personalized keycaps.

## Design Principles

1. **Minimal & Elegant**: Clean interfaces, generous whitespace
2. **Dark & Sophisticated**: Deep blacks (#050505, #0a0a0a) with vibrant cyan (#00ffcc) accents
3. **Highlight Customization**: Every page should showcase the power of personalization
4. **Jaw-Dropping**: Premium 3D elements, smooth animations, stunning visuals
5. **Clear Hierarchy**: Guide users naturally through the experience

## Technical Stack

- **Frontend**: Vite + React
- **3D**: React Three Fiber + Three.js
- **Animations**: GSAP + ScrollTrigger
- **Styling**: Tailwind CSS
- **State**: Zustand
- **Backend**: Supabase
- **Page Generation**: Stitch MCP

## Key Features

- Full 3D keyboard configurator with layer management
- Texture upload and manipulation
- Real-time preview with realistic lighting
- Multi-page site with consistent design system
- Scroll-based animations
- E-commerce integration
